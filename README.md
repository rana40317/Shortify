Project Title: Shortify: A URL Shortening web application.
Project Description: A URL Shortener is a web application that converts long, complex URLs into short, easy-to-share links. When a user visits the short link, they are automatically redirected to the original URL.
Problem Statement: In today's digital world, URLs are often long, complex, and difficult to share. Long URLs are hard to remember, look unprofessional, and sometimes break when shared via email or social media. There is a need for a simple web-based system that can convert lengthy URLs into short, clean, and shareable links while also tracking their usage.
Feature of the project: In the current version of the project, the system will generate a short URL and return a clean URL immediately to the user.
Technology Slack: 
        a) Frontend
            i) HTML: To create the structure of the web page.
            ii) CSS: To give styling and layout of the structured web page.
            iii) JavaScript: Copy to clipboard, form validation
            iv) Bootstrap: Responsive design framework
            v) Jinja: Python template engine for dynamic pages
        b) Backend
            i) Python: It is a core programming Language.
            ii) Flask: It is a web framework to handle routes and requests.
            iii) Random String: Generate unique shortcodes for a convenient URL.
Future Improvements: It needs to tie up with a database to store URL and use Docker for scalability of the web application.
